# Contributing to Forklift Technical Manifest

First off, thanks for taking the time to contribute! 🔧

This project was built by a maintenance technician to solve real problems on the shop floor. Contributions from fellow techs are especially welcome.

## Ways to Contribute

### 🐛 Report Bugs
Found something broken? [Open an issue](https://github.com/TheWiredWatchman/forklift-technical-manifest/issues) with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Screenshots if applicable

### 📝 Improve Documentation
- Fix typos
- Clarify confusing sections
- Add examples

### ⚠️ Add Error Codes
Know an error code that's missing? Include:
- The error code
- Plain-English description
- Common causes
- First things to check

### 🔧 Add Features
Before starting major work, open an issue to discuss. Some ideas:
- Support for other forklift models
- Additional PM intervals
- Integration with maintenance management systems

## Pull Request Process

1. Fork the repo
2. Create a branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Test that the HTML still loads and works
5. Commit with clear messages
6. Push and open a PR

## Code Style

This is mostly HTML/CSS/JS in a single file. Keep it:
- Readable
- Well-commented where logic is complex
- Consistent with existing style
